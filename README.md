# Installation
```bash
git clone https://github.com/Renjuju/Checkers.git
```
Go to the Checkers directory
```bash
npm install
```
Then start up the server
```bash
node server.js
```
The site will be up at localhost

#Deployment to server
```bash
npm run deploy
```

#Unit tests!
```bash
npm test
```
